# Security Policy

## 지원 대상

현재 보안 안내는 GitHub Releases에 등록된 최신 버전의 HD2 Crash Radar를 기준으로 합니다.

공식 배포 위치:

https://github.com/SAM7599/HD2-Crash-Radar/releases

출처가 불분명한 재배포 파일, 수정된 파일, 압축을 다시 묶은 파일은 지원 대상이 아닙니다.

## 보안 원칙

HD2 Crash Radar는 헬다이버즈2 크래시/튕김 원인 확인을 돕기 위한 비공식 진단 도구입니다.

이 도구는 다음 행위를 하지 않습니다.

- 게임 파일 변조
- GameGuard/nProtect 우회
- 메모리 주입
- DLL 삽입
- 프로세스 후킹
- 자동 입력/매크로/치트 기능 제공
- 오류 보고서 자동 업로드
- 사용자 정보 자동 전송
- 백그라운드 자동 통신

## 로컬에서 확인하는 정보

진단을 위해 사용자의 PC에서 다음 정보를 읽을 수 있습니다.

- Windows 이벤트 로그
- 헬다이버즈2 설치 경로
- GameGuard/nProtect 관련 로컬 단서
- HD2 `shader_cache` 경로
- Steam `shadercache\553850` 경로
- `user_settings.config` 존재 여부
- 그래픽/DX/오버레이 관련 Windows 오류 단서

이 정보는 사용자의 PC 안에서만 분석됩니다.

## 온라인 기능

온라인 기능은 사용자가 버튼을 눌렀을 때만 동작합니다.

현재 온라인 기능:

- GitHub Releases 최신 버전 확인

확인 대상:

```text
https://github.com/SAM7599/HD2-Crash-Radar/releases
```
이 기능은 GitHub 공개 릴리스만 읽습니다.

다음 정보는 온라인으로 전송하지 않습니다.
- 오류 보고서
- 사용자 이름
- PC 이름
- 사용자 폴더 경로
- IP 주소
- Steam ID
- 게임 설치 경로
- Windows 이벤트 로그 내용

## 보고서 저장과 개인정보
`공유용 저장`기능은 사용자가 직접 실행했을 때만 보고서 파일을 생성합니다.

보고서는 자동 전송되지 않습니다.
사용자가 직접 저장하고 직접 공유하는 방식입니다.
공유용 보고서에는 개인정보 노출을 줄이기 위해 다음 항목을 마스킹합니다.
- 사용자 이름
- PC 이름
- 사용자 폴더 경로
- IP 주소
- 이메일 주소
- Steam ID
- 토큰/비밀번호/API 키 형태 문자열
단, 모든 개인정보를 완벽하게 제거한다고 보장할 수는 없습니다.
보고서를 공개 커뮤니티에 올리기 전에는 내용을 직접 확인하는 것을 권장합니다.

 ## 캐시 정리 기능
 캐시 정리 기능은 안전 확인을 통과한 경로에서만 동작합니다.
 
대상 경로:

`%APPDATA%\Arrowhead\Helldivers2\shader_cache
steamapps\shadercache\553850`

정리 방식:
- 폴더 자체는 삭제하지 않습니다.
- 내부 파일만 삭제합니다.
- 다른 게임의 Steam shadercache 폴더는 대상으로 하지 않습니다.
`user_settings.config`삭제 기능은 설정 초기화가 발생할 수 있으므로 마지막 수단으로 사용해야 합니다.

## 취약점 제보
보안 문제나 개인정보 노출 가능성을 발견했다면 GitHub Issues를 통해 제보해 주세요.

제보 시 포함하면 좋은 정보:
- 사용한 HD2 Crash Radar 버전
- Windows 버전
- 문제가 발생한 기능
- 재현 방법
- 민감정보를 제거한 스크린샷 또는 설명

주의:
- 실제 개인정보, 토큰, 비밀번호, Steam ID가 포함된 원본 로그는 공개 이슈에 올리지 마세요.
- 필요한 경우 민감정보를 마스킹한 뒤 공유해 주세요.

## 면책 고지
HD2 Crash Radar는 Arrowhead Game Studios, PlayStation, Steam, Discord와 공식 관련이 없는 비공식 팬 제작 도구입니다.
진단 결과는 확정 판정이 아니라 원인 후보를 좁히기 위한 참고 정보입니다.
사용자는 도구 사용 전후의 변경 사항을 직접 확인해야 합니다.
